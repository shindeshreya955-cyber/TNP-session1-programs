# T&P Technical Training Practice Programs 🚀

Welcome to this repository!

This repository contains all practical programs performed during the T&P (Training and Placement) Technical Training session conducted by GIRI Tech Hub Pvt. Ltd. on 18 May 2026.

The main objective of this session was to build a strong foundation in programming concepts, improve logical thinking, and develop problem-solving skills through hands-on coding practice.

Although the concepts covered were based on Java programming, the practical implementation and coding practice were performed using Python in IDLE for better understanding and execution.

---

# 📚 Topics Covered

## Basic Programming Concepts

* Displaying messages
* Input and output operations
* Understanding basic data types
* Arithmetic operations

## Mathematical Programs

* Percentage calculation
* Simple Interest calculation
* Swapping two numbers
* Triangle angle calculation
* Time conversion (seconds to hours, minutes, seconds)

## Logical and Conditional Programs

* Even or Odd number check
* Positive, Negative or Zero check
* Divisibility check
* Neon Number program

## ASCII and Character Operations

* ASCII value of character
* Character case conversion using ASCII values

## Triangle Programs

* Triangle validation
* Equilateral, Isosceles and Scalene triangle identification

## Employee Salary Program

* Gross salary calculation using HRA and DA
